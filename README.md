### 📖 Theory & Code

- Working with Large Language Models (LLMs)
- Prompt Engineering

- Interactive AI Applications using Gradio
- Mini Projects

# 🛠 Prerequisites

Before getting started, make sure you have:

- Python 3.11+
- Git
- Vs Code
- An API Key (Groq API / OpenRouter API )

---

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/sagar-tandan-ai/Generative-AI-using-LangChain.git
````

Move into the project folder.

```bash
cd Generative-AI-using-LangChain
```

---

## 2. Install UV (Recommended)

### Windows

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### macOS / Linux

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Verify installation

```bash
uv --version
```

---

## 3. Create a Virtual Environment

```bash
uv venv
```

Activate it.

### Windows

```bash
.venv\Scripts\activate
```

### macOS/Linux

```bash
source .venv/bin/activate
```

---

## 4. Install Dependencies

```bash
uv sync
```

---

## 5. Configure Environment Variables

Create a `.env` file from the provided example.

### Windows

```bash
copy .env.example .env
```

### macOS/Linux

```bash
cp .env.example .env
```

Open the `.env` file and add your API key.

Example:

```env
GROQ_API_KEY=your_google_api_key

# OR

OPENROUTER_API_KEY=your_openrouter_api_key
```
